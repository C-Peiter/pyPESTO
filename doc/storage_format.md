# HDF5 data format

## Parameter estimation results
Parameter estimation results are saved in `/optimize_results/`.

### Results per local optimization
```
+ /optimization_results/multistarts/$n/
  - fval: ...
  - x: ...
  - grad: ...
  - hess: ...
  - n_fval: ...
  - n_grad: ...
  - n_hess: ...
  - n_res: ...
  - n_sres: ...
  - x0: ...
  - fval0: ...
  - trace: ...
  - exitflag: ...
  - time: ...
  - message: ...
```
